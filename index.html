Crystal Game Collector

Instructions

In this game you will try and match the "Goal" number. The way you will reach is by clicking on the various gems. Each gem has a specific value. You will only find out the value of each gem after clicking on it the first time. Choose which gems you to add in order to reach the Goal needed. If you match it, you win! If you go over,sorry try again!
